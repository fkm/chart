# @fkm/chart
Chart widget built around D3.

## Usage
* `npm install --save-dev @fkm/chart`
* `import Chart, { getLineSet } from '@fkm/chart'`
* `let datasets = [getLineSet(data)];`
* `let chart = new Chart(config);`
* `chart.setDatasets(datasets);`
